Why do the Moomins use semantic html?

Semantics are the meaning of a word, phrase or text. Moomins love friends and family and find meaning in everything, even the smallest of things. HTML5 semantic tags aim to make our code more descriptive, more meaningful.
See below for a list of reasons.

  • Better for SEO optimisation as search engines will consider semantic content as import keywords to influence pag ranking
  • Screen readers use it as signposts for those with visual impairment
  • Makes code more readable
  • Mirrors proper custom element/component naming
